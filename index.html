Help & Documentation

Guides and troubleshooting

Quick Start Guide

1. Adding Events: Go to Events Manager → Add Event → Fill in details → Save

2. Recording Results: Events Manager → Select completed event → Enter scores → Mark verified

3. Blog Posts: Blog Manager → New Post → Write content → Publish

4. Weekly Winners: Weekly Winners → Add Winner → Select athlete → Feature

5. Gallery: Gallery Manager → Add Photo → Enter URL → Save

Troubleshooting

Content not showing: Check status is set to "published"

Images not loading: Verify URL is valid and accessible

Standings not updating: Ensure results are marked as verified

Login issues: Clear browser cache, try different browser
